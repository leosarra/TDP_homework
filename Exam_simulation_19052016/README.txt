=============================================================================
                            ISTRUZIONI IMPORTANTI

   1. Inserire i propri dati nella GUI
   2. Rispondere con la GUI alle domande contenute nel file domande{nn}.html 
   3. Salvare e chiudere la GUI
   4. Leggere il testo dell'esercizio nel file testo.html
   5. Svolgere gli esercizi senza modificare la struttura delle directory
   6. Svolgere ciascun esercizio nella directory esercizio{i}
   7. Compilare con il comando make
   8. Usare il comando ./test_esercizio{i} per eseguire il programma di prova
   9. Eseguire il comando TDP_upload_compito.bash per consegnare il compito   
=============================================================================
