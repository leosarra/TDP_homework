Domande di esame - Compito 4

Domanda 1

Si consideri il seguente frammento di codice: 
double x=5.5;
{
int x = (int)x;
}
printf("%f",x);
Quale delle seguenti affermazioni è corretta?
  • A Si verifica un errore in fase di esecuzione in quanto l'operazione di cast da double a int è errata.
  • B Si verifica un errore di compilazione in quanto la variabile x è definita due volte.
  • C L'istruzione printf stampa 5.
  • D L'istruzione printf stampa 5.5.

Domanda 3

Si consideri la seguente funzione: 
void f(int **mat) {
    ...
}
Una sola delle seguenti affermazioni è vera. Quale?
  • A La funzione può essere invocata passando come argomento una qualsiasi matrice (allocata staticamente o dinamicamente), ma non è in grado di fare operazioni sulla matrice in quanto non sono definite le sue dimensioni. mat non ` inizializzata.
  • B La funzione può essere invocata passando come argomento una qualsiasi matrice (allocata staticamente o dinamicamente) e può eseguire qualsiasi operazione sulla matrice. mat.
  • C La funzione può essere invocata passando come argomento una matrice allocata dinamicamente, ma non è in grado di fare operazioni sulla matrice in quanto non sono definite le sue dimensioni.
  • D La funzione può essere invocata passando come argomento una matrice allocata dinamicamente e può eseguire qualsiasi operazione sulla matrice.

Domanda 4

Si considerino le seguenti definizioni: 
typedef int TipoInfoSCL;

struct ElemSCL{
  TipoInfoSCL info;
  struct ElemSCL* next;
};

typedef struct ElemSCL TipoNodoSCL;
typedef TipoNodoSCL * TipoSCL;

void cosaFaSCL(TipoSCL * scl, TipoInfoSCL e) {
	*scl = (TipoNodoSCL*) malloc(sizeof(TipoNodoSCL));
	(*scl)->info = e;
	(*scl)->next = NULL;
}
ed il seguente frammento di programma: 
scl1 = NULL; 
cosaFaSCL(&scl1,3);
cosaFaSCL(&scl1,5);
Una sola delle seguenti affermazioni è corretta. Quale?
  • A Il frammento di programma provoca un errore a tempo di esecuzione.
  • B Il frammento di programma viene eseguito correttamente e la SCL scl1 rappresenta la lista [3, 5].
  • C Il frammento di programma viene eseguito correttamente e la SCL scl1 rappresenta la lista [5].
  • D Il frammento di programma viene eseguito correttamente e la SCL scl1 rappresenta la lista [5, 3].
